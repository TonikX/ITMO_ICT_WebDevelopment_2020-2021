#  Создание нового воина


**URL** : `war/warriors/create`

**Method** : `POST`

**Data constraints** : `{}`

**Content-Type** : application/json

**Request data** :
```json
{
    "race": "s",
    "name": "Сергей",
    "level": 0,
    "profession": 3
}
```

**Response data** :
```json
{
    "id": 6,
    "skills": [],
    "race": "s",
    "name": "Сергей",
    "level": 0,
    "profession": 3
}
```

### Добавить данные о паре 

**URL** : `/pair/create/`

**HTTP 201 Created**

**Allow** : POST, OPTIONS

**Content-Type** : application/json

**Vary** : Accept

**JSON** :
```json
{
    "id": 2,
    "group": "1",
    "pair_number": 2,
    "name_day": "mon",
    "room": 211,
    "teacher": 1,
    "subject": 3
}
```

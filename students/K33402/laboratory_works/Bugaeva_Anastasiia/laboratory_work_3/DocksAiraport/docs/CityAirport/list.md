# List of all CityAirport

Выводит список городов и их аэропортов

**URL** : `/cityAirport`

**Method** : `GET`

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
[
    {
        "id": 1,
        "cityName": "Москва",
        "airportName": "Домодедово",
    }
]
```

# List of all FlightAttendant

Выводит информацию обо всех Бортпроводниках

**URL** : `/FlightAttendant/list/`

**Method** : `GET`

**Data constraints** : `{}`

## Success Responses

**Code** : `200 OK`

**Content** : `{[]}`

```json
[
  {
    "id": 1,
    "Education": "I",
    "Experience": 2,
    "user": 5,
    "board": [
      1
    ]
  }
]
```

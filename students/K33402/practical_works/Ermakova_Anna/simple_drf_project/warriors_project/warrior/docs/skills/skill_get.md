# Получить инфорацию о навыках

Метод для получения информации о навыках:

**URL** : `/war/skills/`

**Method** : `GET`

**Auth required** : None

**Permissions required** : None

## Success Responses

**Code** : `200 Success`

**Content** : `{[]}`

```json
{
  "Skills": [
    {
      "title": "string"
    }
  ]
}
```

# Получить базовую инфорацию о воине

Метод для получения информации о воине:

**URL** : `/war/warriors`

**Method** : `GET`

**Auth required** : None

**Permissions required** : None

## Success Responses

**Code** : `200 Success`

**Content** : `{[]}`

```json
{
  "name": "String",
  "race": "String",
  "level": "Number"
}
```

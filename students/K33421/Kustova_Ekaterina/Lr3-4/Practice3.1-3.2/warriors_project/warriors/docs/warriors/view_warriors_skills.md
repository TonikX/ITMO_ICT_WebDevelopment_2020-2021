# Список всех воинов и их навыков

###Вывод информации о всех воинах и их навыках

**URL** : `warriors_skills/`

**Method** : `GET`

**Auth required** : YES

**Permissions required** : None

**Data constraints** : `{}`

**Success info** : HTTP 200 OK

**Content-Type** : application/json

**JSON** :
```json
{
    "Skill": [
        {
            "id": 1,
            "skill": ["Healing"],
            "race": "s",
            "name": "Warrior",
            "level": 1
        }
    ]
}
```
